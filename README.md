+ 本地写代码的地方叫做工作区
+ 使用`add`进去的添加的地方叫做暂存区
+ `add`结束，`commit`进去的就是仓库
以上的操作，最终都是在本地的版本库。只有push，才是最终进入远程仓库的结果。
